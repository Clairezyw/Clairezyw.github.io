生命周期的介绍

react生命周期分为三个阶段：初始化阶段、运行阶段和销毁阶段。这三个阶段中分别运行不同的函数，下面我们来详细看一下包含的函数

初始化阶段

  • getDefaultProps：只调用一次，用于实例之间共享内存，注意该方法在React.createclass之后接着调用，也就是说调用该方法时实例尚未初始化。
  • getInitialState：初始化每个实例特有的状态
  • componentWillMount：render之前最后一次修改状态的机会
  • render: 只能访问this.props和this.state，只要一个顶层组件，且不允许修改状态和dom的输出
  • componentDidMount：render成功之后渲染真实的dom之后触发，此时可以修改真实的dom

运行阶段

  • componentWillReceiveProps: 父组件修改属性触发，可以修改新属性
  • shouldComponentUpdate:是否更新，若返回false则会阻止render的调用
  • componentWillUpdate：不能修改属性和方法
  • render：和初始化阶段相同
  • componentDidUpdate：此阶段可修改dom
