WebRTC简介

Bringing real-time communications to the web
Building a state-of-the-art media stack into Chrome
Developing a new communications platform

简单来说，WebRTC（Web Real-Time Communication）是一种可以支持浏览器实时语音或视频通信的API
agora-sdk-web就是基于WebRTC技术的

具体使用

下载
yarn add agora-rtc-sdk --dev

初始化client

import { AgoraRTC } from 'agora-rtc-sdk'

 const client = AgoraRTC.createClient({ mode: 'live', codec: 'h264' })

 client.init(APP_ID, () => {
    console.log('init success')
    client.join(channelKey, channelName, uid, (userId) => {
      rtcParams.uid = uid
      console.log(`join channel: ${channelName} success, uid: ${userId}`)
    }, (err) => {
      console.error('client join failed', err)
    })
  }, (err) => {
    console.error(err)
  });

订阅远端视频流

   // 添加视频流
  client.on('stream-added', (evt) => {
    const remoteStream = evt.stream;
    const id = remoteStream.getId();
    if (id !== rtcParams.uid) {
      client.subscribe(remoteStream, (err) => {
        console.log('stream subscribe failed', err)
      })
    }
    console.log('stream-added remote-uid: ', id)
  })

  // 订阅视频流
  client.on('stream-subscribed', (evt) => {
    const remoteStream = evt.stream
    const id = remoteStream.getId()
    remoteStreams.push(remoteStream)
    remoteStream.play('videoContainer', { fit: 'cover' }, (errState) => {
      if (errState) {
        document.getElementById('play').addEventListener('click', () => {
          remoteStream.play('videoContainer')
        })
      }
    })
    console.log('stream-subscribed remote-uid: ', id)
  })

  client.getRemoteVideoStats((remoteVideoStatsMap) => {
    for (const uid in remoteVideoStatsMap) {
      if (remoteVideoStatsMap[uid]) {
        if (remoteVideoStatsMap[uid].TransportDelay) {
          data.videoDelay = remoteVideoStatsMap[uid].TransportDelay
        }
        if (remoteVideoStatsMap[uid].PacketLossRate) {
          data.videoLost = remoteVideoStatsMap[uid].PacketLossRate

  client.getRemoteAudioStats((remoteAudioStatsMap) => {
    for (const uid in remoteAudioStatsMap) {

        }
        if (remoteAudioStatsMap[uid].PacketLossRate) {
          data.audioLost = remoteAudioStatsMap[uid].PacketLossRate

遇到问题

web端黑屏但是安卓端却可以看到直播？

PC 或移动端用户（使用 Agora Native SDK 的用户）在直播场景下必须调用 enableWebSdkInteroperability 打开互通功能。

chrome浏览器刷新后黑屏？

根据chrome autoplay政策,chrome 70+的浏览器无法使用自动播放功能，只有在用户手势触发之后才会自动播放，目前Stream.play，Stream.startAudioMixing 以及 Stream.getAudioLevel 这三个方法必须由用户手势触发

使用感受

  • 卡顿时间相较于原始的推流减少了很多
  • UI方面可自定义实现，避免了重写video组件样式的痛点
  • 实现起来非常简单,官方提供多种api
  • 文档和社区不是很完善, 遇到问题解决起来相对比较困难
