let命令

Veröffentlicht am 2017-12-11 | in ECMAScript . let | 浏览 次
1、let声明：

let命令在声明时，只在let所在的代码块中有效，看下面例子：

{  var a =100;
   let b =200;
}
console.log(a);//100
console.log(b);//undefined
2、不存在变量提升
var a =[];
for (var i=0; i<10;i++){
  var c =i;
  a[i]=function(){
    console.log(c);
  }
}
a[5]();//输出结果为9

这是因为i在for循环内部进行递增，当for循环体的外部调用i时，i必须递增完成后才能调用。所以输出9

 //运用let
 var a=[];
 for(var i =0; i<10; i++){
   let c =i;
   a[i]=function(){
     console.log(c);
     }
 }
 a[5]();//输出结果为5

从上面的例子可以看出let有点类似于java中的局部变量。

3、暂时性死区

所谓的暂时性死区就是在变量声明之前调用他，结果会返回undefined，不仅let有这个特性，var也有

 console.log(a);//undefined
 let a =1;
4、不可重复声明

与java语言类似，使用let定义变量时，变量名不可重复

{ var a =100;
  var a =200;
}
console.log(a);//200 var声明变量时，最新声明的a的值会覆盖之前声明的a的值，所以不会报错

{
  var b =100;
  let b = 200;
}
console.log(b);//error 重复声明
{
  let c = 100;
  let c =200;
}
console.log(c);//error 重复声明

但在不同的代码块中，可以重复声明

 var a =100;
 function test(b){
  let a = b;
 }//不会报错
