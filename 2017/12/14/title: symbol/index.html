symbol

在es6中symbol是被引入的一种新的原始数据类型，表示独一无二的值，在下面的例子中symbol只是对当前symbol值的描述，即使symbol的参数相同，symbol 的值也不相同。

let name =Symbol('name');
let name1 =Symbol('name');
console.log(name==name1);//false

symbol常用来模拟私有属性，看下面的例子

let getName = Symbol('getName');
 module.exports=class User{
     [getName](){
         return 'stalary';
       }
       print(){
       console.log(this[getName]);}
     }
 }

在另一个页面中我们调用一下user的方法

 const User =require('./User');//引入User类
let getName = Symbol('getName');
let user =new User();
user.print();//调用print方法，输出stalary
user[getName]();//调用user的getame方法输出is not a function

上述例题说明symbol定义的属性类似于私有属性，不能被其他类调用，此外需要注意的是使用symbol定义的方法时不可以使用点连接只能使用中括号。

属性名的遍历

先看一个例子

let name =Symbol('name');
 var obj ={
   age:22;
   [name]:'claire';
 }
 //第一种使用Object.keys方法遍历obj属性
 console.log(Object.keys(obj));//age
 //第二种方法
 console.log(Object.getOwnPropertyNames)//'age'
 //第三种方法使用for in遍历
 for(let i in obj){
 console.log(i);
 }//age
 //第四种方法
 console.log(Object.getOwnPropertySymbols(obj));// 输出Symbol(name) 
//第五种方法
console.log(Reflect.ownKeys(obj));//'age', Symbol(name)

我们可以看出前三种普遍使用的遍历属性的方法都无法遍历到symbol定义的属性，而第四种方法，只能遍历symbol属性，第五种是新引入的api既可以遍历到常规属性也可以遍历到symbol属性
下面我们来看一下symbol和遍历器的使用

let arr =[1,2,3];
var iterator =arr[Symbol.iterator]();
console.log(iterator.next());
console.log(iterator.next());
console.log(iterator.next());
console.log(iterator.next());
/*输出
{ value: 1, done: false }
{ value: 2, done: false }
{ value: 2,3done: false }
{ value: undefined, done: true }
done表示是否遍历完成，若遍历完成显示true，否则为false
*/

通过上面的例题，我们再来深入了解一下symbol的遍历

 class User{
   constructor(users){
      this.users = users;
    }
    //iterator
    [Symbol.iterator](){
      let i =0;
      let self =this;
      const names = Object.keys(this.users);
      const len =names.length;
      return {
         next:function(){
            let name =names[i++];
            let age = self.users[name];
            return {
             value:{name:name,age},done :i>len
            }

         }
      }
    }
 }
 let user = new User('stalary':21,'claire':20);
 let iterator = user[Symbol.iterator]();
console.log(iterator.next());
console.log(iterator.next());
console.log(iterator.next());
/*
{ value: { name: 'stalary', age: 21 }, done: false }
{ value: { name: 'claire', age:20 }, done: false }
{ value: { name: undefined, age: undefined }, done: true }

*/
