classList属性

Veröffentlicht am 2017-12-07 | in javascript . classList | 浏览 次

在操作类名时，需要通过className进行添加、修改、删除操作，由于classnName是字符串，所以即使修改一部分类名，也要重新设置className的值，看下面的例子👇

 <div class ="user bd time">...</div>

//删除user类
//首先获取div类名，然后按空格分割成数组
var arr = div.className.split(/\+s/);
var pos = 0,i,len;
for(i = 0,len= arr.length;i<len;i++){
   if (arr[i]=="user"){
   //保存user的位置
        pos = i;
        break;}
}
//删除user
arr.splice(pos,1);
//将剩下的类名拼接起来
div.className = arr.join(" ");

html5新增了操作类的属性：

  • add（value）：将给定的字符串添加到列表中，若存在则不添加
  • contains（value）：判断是否包含给定的值，若包含返回true否则返回false。
  • remove（value）：将给定的字符串从列表中删除
  • toggle（value）：如果列表中存在给定的字符串则删除它，否则添加它；
    因此，上述代码就可以简化为
div.classList.remove("user");

//添加类名temp
div.classList.add("temp");
//判断是否包含bd类
div.classList.contain("bd");//true
//切换user类
div.classList.toggle("user");
