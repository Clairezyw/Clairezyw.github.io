回调函数

    function fn1(callback) {
        setTimeout(function() {
            // 执行f1的任务代码
            callback();
        }, 1000)
    }

回调函数的优点是简单、容易部署，先执行主要的代码，耗时的代码延迟执行，缺点是代码之间高度耦合并且只执行一个回调函数；

http重点总结

Veröffentlicht am 2018-03-09 | in 网络基础 . http重点总结

1、http的主要特点

  • 简单快速：给定url地址返回相应的资源
  • 灵活：同一个http 协议可以返回不同的资源
  • 无连接：每次请求和响应完成后http自动断开连接
  • 无状态：http不保存请求和响应之间的状态，也就是说同一个用户请求两次，http无法识别这两次请求是否是同一个发送的。

项目总结

前几日大概准备的十天的期末考试，前端的学习进程有些耽误了。近几日做了一个小项目，加深了一下对vue的理解，但也爬了不少坑😂，写完之后，及时总结一下遇到的诸多小问题…

1、在项目搭建过程，使用npm install命令下载过慢

解决办法：使用淘宝镜像，附上淘宝镜像的网址，http://npm.taobao.org/ 在使用npm命令的淘宝镜像时就可以在终端输入

npm config set registry https://registry.npm.taobao.org

运行之后可以检查一下是否成功获取淘宝镜像

npm config get registry
